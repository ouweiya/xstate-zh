# Stately Studio 团队计划

[升级到团队账户](https://stately.ai/pricing) 以享受我们所有的专业功能以及许多旨在帮助您与团队在任意数量项目上协作的功能。团队账户包括以下功能：

## 团队功能

- [专业计划](studio-pro-plan.mdx)中的所有内容
- 根据您的需求创建一个或多个[团队](teams.mdx)
- [私人、共享团队项目](/docs/projects/#change-a-projects-visibility)
- 添加最多10个[团队成员](/docs/teams/#invite-new-members-to-a-team)
- [团队管理员和编辑](/docs/teams/#admin-role)
- 授予团队项目的[仅查看访问权限](/docs/teams/#viewer-role)
- 实时协作（即将推出！）

我们即将推出更多团队功能。请求功能并查看我们在[路线图](https://feedback.stately.ai)上的计划。

我们在所有高级计划上提供**免费试用**，以便您探索我们的团队功能如何为您和您的团队工作。

:::studio

**Stately Studio 将始终对免费计划的社区用户免费**，并且我们将在每个计划中提供许多未来功能。

:::

## 团队账户

您可以选择所有高级计划的月度或年度计费。

:::studio

### 作为团队成员的专业功能

如果您是其他用户[团队](teams.mdx)的团队成员，担任[管理员](/docs/teams/#admin-role)或[编辑](/docs/teams/#editor-role)角色，您也可以访问[高级功能](studio-pro-plan.mdx)！但是，如果您想创建自己的团队，您需要[升级到自己的专业或团队订阅](upgrade.mdx)。

:::

### 如何注册

您可以从[**登录页面**](https://stately.ai/registry/login)或[编辑器](https://stately.ai/editor)顶部栏的**登录**按钮注册 Stately Studio 账户。

阅读更多关于[注册 Stately Studio](sign-up.mdx)的信息。

### 如何升级

当您登录 Stately Studio 时，可以使用编辑器底部的**升级**按钮进行[升级](upgrade.mdx)。

## 优先支持

当您注册付费计划时，您将收到一封电子邮件，包含访问 Stately 团队优先支持的信息。

## 取消您的团队计划

您可以随时取消您的团队计划。当您取消团队计划时：

- 您的项目将变为只读。
- 无法再创建新的团队项目。
- 无法再将团队成员添加到您的团队。
- 账户所有者可以删除团队。
- 账户所有者可以将项目从他们的团队移动到“我的项目”，除非账户所有者有专业订阅，否则这些项目将变为公开。
- 团队成员的私人项目将变为只读。
- 团队成员将失去对专业功能的访问权限，除非他们购买了自己的专业订阅。

:::info

## 为什么要支付高级计划费用？

我们的高级计划包括私人和未列出的项目、Stately AI 的 AI 辅助、Stately Sky 的实时工作流、团队管理的有用功能等等。您的订阅还帮助支持 Stately 团队构建和改进 Stately Studio 和 XState。

:::
