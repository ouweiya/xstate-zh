**使用 AI 生成** 是一个实验性功能，可以帮助你从文本描述中自动创建机器。你可以为新机器生成一个流程，或者使用流程描述来描述你想要如何修改当前流程。

:::tip

社区用户每月可以尝试有限次数的生成流程。[升级你的现有计划](https://stately.ai/registry/billing) 以获得更多的生成次数。

:::

:::studio

**使用 AI 生成** 是 Stately Studio 的高级功能。你可以通过免费试用来尝试 Stately Studio 的高级计划。[查看我们的专业计划功能](studio-pro-plan.mdx)、[团队计划](studio-team-plan.mdx)、[企业计划](studio-enterprise-plan.mdx) 或 [升级你的现有计划](https://stately.ai/registry/billing)。

:::

你每月可以使用有限次数的生成。可用的生成次数在每月初重置。

你可能想要生成流程的原因有很多，包括：

- 你有一个复杂的机器想法，想要快速开始
- 你想要一个特定流程的状态机示例
- 你是新手，想要了解如何建模状态机

## 生成新流程

生成新流程将用从你的文本描述中创建的全新流程覆盖你当前的机器。

1. 在创建新机器时选择 <Sparkles size={18} /> **使用 AI 生成** 或使用画布工具中的 <Sparkles size={18} /> 闪光图标按钮打开生成流程对话框。
2. 输入你想要生成的流程的详细文本描述，并使用 **生成** 按钮生成你的流程。

当你生成全新流程时，只有你的提示文本会发送到 OpenAI。

## 从当前流程生成

1. 使用画布工具中的 <Sparkles size={18} /> 闪光图标按钮打开生成流程对话框。
2. 确保选择了 **从当前流程生成** 以修改你的当前机器。
3. 输入你想要生成的流程的详细文本描述，并使用 **生成** 按钮生成你的流程。

当你使用 **从当前流程生成** 时，我们会与 OpenAI 共享你的机器定义以帮助生成响应。

## 查看历史记录

你可以查看用于生成当前机器流程的文本描述历史记录。使用 **生成流程** 对话框中的 **查看历史记录** 按钮查找你的 **提示历史记录**。

- 使用 **打开流程** 预览从所选提示生成的[版本](versions.mdx)。
- 使用 **复制到提示** 将所选提示复制到 **生成流程** 对话框的文本描述区域。

每次生成流程时都会自动保存一个新[版本](versions.mdx)。使用 <History size={18} /> **版本** 面板并打开 **显示自动保存的版本** 切换开关以浏览当前机器的自动保存版本。
