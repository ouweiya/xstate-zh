## 浅色模式和深色模式

您可以选择偏好浅色模式或深色模式，Stately Studio 会记住您的设置。您必须登录才能选择您的偏好模式。

在文档中切换浅色和深色模式设置，以预览上面的机器在 Stately Studio 中切换浅色和深色模式。

### 切换浅色和深色模式

1. 使用 Stately Studio 右上角的头像按钮打开用户菜单。
2. 使用设置来 <Sun size={18}/> **切换到浅色模式** 或 <Moon size={18}/> **切换到深色模式**。

:::studio

我们计划在未来根据您的操作系统偏好设置浅色模式和深色模式。如果这是您的优先事项，请[在我们的路线图中为此功能投票](https://feedback.stately.ai/editor/p/automatic-lightdark-mode-based-on-system-preferences)。

## 透明度

在 Stately Studio 画布中，事件是半透明的，以便更容易区分重叠的事件和转换。如果您愿意，可以选择关闭透明度。

### 启用和禁用透明度

1. 使用 Stately Studio 右上角的头像按钮打开用户菜单。
2. 使用设置来 <Sparkle size={18}/> **禁用透明度** 或 <Sparkle size={18}/> **启用透明度**。

Safari 尚不支持透明度。
