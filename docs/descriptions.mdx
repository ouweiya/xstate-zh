title: 描述
---

你可以为状态和事件节点添加描述，以描述其目的并与团队共享相关备注。描述支持 markdown 格式，包括链接和图像。

每个状态或事件只能添加一个描述。当你将状态图导出为 JSON 时，机器对象会在状态或事件的 `description` 中包含你的描述。

## 为状态、事件或转换添加描述

使用 <Plus size={18} /> 加号图标按钮，并从菜单中选择 <AlignLeft size={18} /> **描述**，以向选定的状态、事件或转换添加描述块。

你也可以从右侧工具菜单中的 <Info size={18} /> **详细信息** 面板中添加描述。

## 删除状态、事件或转换的描述

选择要删除描述的状态、事件或转换，打开 <Info size={18} /> **详细信息** 面板，并删除描述文本区域的内容。
