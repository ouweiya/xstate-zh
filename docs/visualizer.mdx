# 可视化工具（旧版）

:::studio

您在寻找 _Stately Studio_ 可视化 _编辑器_ 吗？请查看 [Stately Studio 概述](/)。

:::

![Stately 可视化工具的分屏显示。左侧显示了视频播放器的可视化状态机。右侧显示了用于创建可视化状态机的 JavaScript。](/assets/tools/visualizer/visualizer.png)

[Stately 可视化工具](https://stately.ai/viz) 是一个用于创建和检查状态图的工具，可以用来可视化您的应用程序状态。您可以将其用作探索 XState 功能的游乐场。

:::warning

XState 可视化工具已被弃用，不再维护。

:::

我们建议您仅在需要执行操作和守卫或同时可视化多个状态机时继续使用旧版可视化工具。这些功能很快将在 [Stately 编辑器](studio.mdx) 中推出。

## 使用可视化工具

- [带我去可视化工具](https://stately.ai/viz)
- [在检查模式下试用可视化工具](https://stately.ai/viz?inspect)。

可视化工具已被弃用，我们鼓励您将项目迁移到 [Stately 编辑器](studio.mdx)。
