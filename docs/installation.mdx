# 安装

XState **零依赖**，可以在任何运行 JavaScript 的地方运行。你可以使用你喜欢的包管理器来安装 XState，比如 [npm](https://www.npmjs.com/)、[pnpm](https://pnpm.js.org/) 或 [yarn](https://yarnpkg.com/)。

<Tabs>
<TabItem value="npm" label="npm">

```bash
npm install xstate
```

</TabItem>

<TabItem value="pnpm" label="pnpm">

```bash
pnpm install xstate
```

</TabItem>

<TabItem value="yarn" label="yarn">

```bash
yarn add xstate
```

</TabItem>
</Tabs>

## CDN

你也可以从各种 CDN 导入 XState：

- [esm.run](https://esm.run):

```ts
import { createMachine, createActor } from 'https://esm.run/xstate';
```

- [esm.sh](https://esm.sh):

```ts
import { createMachine, createActor } from 'https://esm.sh/xstate';
```

- [Skypack](https://www.skypack.dev):

```ts
import { createMachine, createActor } from 'https://cdn.skypack.dev/xstate';
```
