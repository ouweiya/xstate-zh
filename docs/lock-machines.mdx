---
title: 锁定机器
---

您可以随时锁定机器以防止意外编辑。锁定的机器在编辑器的顶部栏和项目的左侧面板中的机器列表中显示为 <Lock size={18} /> **已锁定** 状态。

将鼠标悬停在 <Lock size={18} /> **已锁定** 文本上可以查看是谁在何时锁定了机器。

:::studio

锁定机器是 Stately Studio 的高级功能。您可以通过免费试用来体验 Stately Studio 的高级计划。[查看我们的专业计划功能](studio-pro-plan.mdx)、[团队计划](studio-team-plan.mdx)、[企业计划](studio-enterprise-plan.mdx)或[升级您的现有计划](https://stately.ai/registry/billing)。

:::

锁定机器会阻止您或您的团队中的任何人执行以下操作：

- 编辑机器
- 删除机器
- 重命名机器
- [导入代码](import-from-code.mdx) 以替换机器
- [恢复机器的过去版本](versions.mdx)

如果在机器被锁定时有人正在查看或编辑该机器，他们的更改将不会被保存，并且他们会收到机器已被锁定的通知。

## 如何锁定和解锁机器

从编辑器菜单中，转到 **机器** > <Unlock size={18} /> **锁定机器** / <Lock size={18} /> **解锁机器**。

从机器的 <Info size={18} /> **详细信息** 面板中，使用 <Unlock size={18} /> 锁定机器 / <Lock size={18} /> 解锁机器图标按钮。

如果机器已经被锁定，您可以通过点击编辑器顶部栏中的 **已锁定** 按钮并确认您希望解锁机器来解锁它。

## 团队角色和锁定机器

所有团队成员都可以查看锁定的机器。只有团队所有者、管理员和编辑者可以锁定和解锁机器。

| 功能                    | 所有者 | 管理员 | 编辑者 | 查看者 |
| ----------------------- | ------ | ------ | ------ | ------ |
| 查看锁定的机器          | ✅ 是  | ✅ 是  | ✅ 是  | ✅ 是  |
| 锁定和解锁机器          | ✅ 是  | ✅ 是  | ✅ 是  | 🚫 否  |
