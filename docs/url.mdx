---
title: 使用共享 URL 共享机器
---

您可以使用机器 URL 与您的团队或公开共享机器。

您的机器只有在以下情况下才可见：

- 项目可见性为**公开**
- 项目可见性为**未列出**
- 项目可见性为**私有**，并且查看者也是您团队的成员

存储在“我的项目”中的私有机器将仅对您可见。

阅读[如何更改项目的可见性设置](projects.mdx#change-a-projects-visibility)。

## 从浏览器地址栏复制 URL

您可以从浏览器地址栏复制机器 URL，这将共享您当前视图和版本中的机器。

## 从共享模式复制 URL

1. 使用 Stately Studio 顶部栏中的**共享**按钮打开共享对话框。
2. 使用**复制共享 URL**按钮将机器的 URL 复制到剪贴板。

## 机器 URL 参数

根据您从哪里复制 URL，它可能包含以下参数：

- **machineId**：机器的唯一 ID。例如，`machineId=491a4c60-5300-4e22-92cf-8a32a8ffffca`。
- **mode**：当前机器模式。例如，`mode=Simulate`。
- **version**：当前[机器版本](versions.mdx)。例如，`version=78730cff-1e90-4190-9e41-426733b71c9f`。
