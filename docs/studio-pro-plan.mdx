# Stately Studio 专业计划

[升级到专业账户](https://stately.ai/pricing) 以创建更具表现力的流程，借助 AI 助手更快地工作，并通过版本控制实现更大的编辑灵活性。专业账户包括以下功能：

## 专业功能

- [社区计划](studio-community-plan.mdx)中的所有内容
- [私人和未列出的项目](/docs/projects/#change-a-projects-visibility)
- [使用 AI 生成和修改](generate-flow.mdx)
- [部署实时工作流](stately-sky-getting-started.mdx)
- [生成 React UI](generate-react.mdx)
- [版本历史](versions.mdx)
- [从 GitHub 导入](import-from-github.mdx)
- [实时模拟模式](live-simulation.mdx)
- [嵌入 Figma 框架](figma.mdx)
- [状态资产](assets.mdx)
- [锁定机器](lock-machines.mdx)
- [生成测试路径](generate-test-paths.mdx)
- [为状态和转换着色](colors.mdx)
- [将流程导出为 markdown](export-as-code.mdx)
- [将流程导出为故事](export-as-code.mdx)
- [优先支持](/docs/studio-pro-plan/#priority-support)
- [GitHub 同步](import-from-github.mdx)
- 演员（即将推出！）

我们还有许多专业功能即将推出。请求功能并查看我们在[路线图](https://feedback.stately.ai)上的计划。

我们在所有高级计划中提供**免费试用**，以便您探索我们的专业功能如何为您工作。

:::studio

**Stately Studio 将始终对我们的社区用户**免费，并且我们将在每个计划中提供许多未来功能。

:::

## 专业账户

您可以选择所有高级计划的月度或年度计费。

:::studio

### 作为团队成员的专业功能

如果您是其他用户[团队](teams.mdx)的团队成员，具有[管理员](/docs/teams/#admin-role)或[编辑](/docs/teams/#editor-role)角色，您也可以访问[高级功能](#pro-features)！但是，如果您想创建自己的团队，您需要[升级到自己的团队订阅](upgrade.mdx)。

:::

### 如何注册

[从注册页面注册 Stately Studio 账户](https://stately.ai/registry/login)或使用[编辑器](https://stately.ai/editor)顶部栏中的**登录**按钮。

阅读更多关于[注册 Stately Studio](sign-up.mdx)的信息。

### 如何升级

当您登录 Stately Studio 时，可以使用编辑器底部的**升级**按钮[升级](upgrade.mdx)。

## 优先支持

当您注册专业计划时，您将收到一封电子邮件，其中包含有关如何从 Stately 团队获取优先支持的信息。

## 取消您的专业计划

您可以随时取消您的专业计划。当您取消专业计划时：

- 新项目将默认公开。
- 私人项目将变为只读。
- 您将无法访问其他专业功能，如使用 Stately Sky 部署实时工作流、AI 生成功能、保存机器版本等。

:::info

## 为什么要支付高级计划费用？

我们的高级计划包括私人和未列出的项目、Stately AI 的 AI 助手、Stately Sky 的实时工作流、团队管理的有用功能等等。您的订阅还帮助支持 Stately 团队构建和改进 Stately Studio 和 XState。

:::
