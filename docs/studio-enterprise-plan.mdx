# Stately Studio 企业计划

升级到企业账户，享受我们所有付费计划的功能，以及为满足您组织的特定需求而设计的功能，包括自定义集成、更高的安全性和顶级支持。企业账户包括以下功能：

## 企业功能

- [团队](studio-team-plan.mdx)计划中的所有内容
- 无限的[生成流程](generate-flow.mdx)
- 灵活的托管
- 专属优先支持
- 自定义服务器位置
- 单点登录 (SSO)
- 审计日志
- 将 Stately 嵌入到您自己的应用中
- 自定义效果集合（操作、参与者等）
- 优先处理的功能请求
- 根据您组织需求定制的计划

我们即将推出更多团队功能。请求功能并查看我们在[路线图](https://feedback.stately.ai)上的计划。

<p><a href="mailto:support@stately.ai?subject=我对 Stately Studio 企业计划感兴趣">发送电子邮件给 Stately 团队</a>，获取根据您组织需求定制的计划。</p>

## 为什么要支付高级计划费用？

我们的高级计划包括私有和未列出的项目、Stately AI 的 AI 辅助、Stately Sky 的实时工作流、团队管理的有用功能等等。您的订阅还帮助支持 Stately 团队构建和改进 Stately Studio 和 XState。
