# 实时模拟模式

实时模拟模式帮助你与团队分享你的模拟机器，而无需屏幕共享或截图。分享生成的实时模拟链接，你的团队可以实时查看你的模拟，甚至可以自己触发事件。

:::studio

实时模拟是 Stately Studio 的高级功能。你可以通过免费试用来体验 Stately Studio 的高级计划。[查看我们的专业计划功能](studio-pro-plan.mdx)、[团队计划](studio-team-plan.mdx)、[企业计划](studio-enterprise-plan.mdx)或[升级你现有的计划](https://stately.ai/registry/billing)。

:::

## 与他人模拟你的机器

你必须登录才能开始实时模拟。从 <Play size={18}/> **模拟** 按钮进入 **模拟** 模式。

:::tip

当你处于 **模拟** 模式时，你的当前状态和下一个可用事件将以蓝色高亮显示，以模拟你的机器的流程。点击可用事件以通过状态转换并检查你的逻辑。

:::

使用左下角的 **实时模式** <Triangle size={18}/> 按钮显示 **实时模拟** 选项，并选择 **开始实时模拟** 以开始实时模拟会话。

当你的机器处于实时模拟模式时，左下角将显示 **实时模拟面板**，其中包含一个六字符的唯一代码（例如：3KNYII），以及 **复制链接** 和 **停止** 按钮。当前状态和下一个事件也将以橙色高亮显示。

使用链接或唯一会话代码与任何人分享你的实时模拟会话。

## 使用唯一会话代码加入实时模拟

1. 从可视化编辑器的中心选择 **模拟** 模式。
2. 使用左下角的 **实时模式** 按钮显示 **实时模拟** 选项。
3. 选择 **加入实时模拟** 以加入现有的实时模拟模式会话。
4. 在文本框中输入你提供的唯一会话代码。
5. 选择 **加入** 按钮以加入会话。

## 使用唯一会话 URL 加入实时模拟

将与你分享的 URL 粘贴到浏览器地址栏并按回车键以加入实时模拟会话。
