# Stately Studio 社区计划

**Stately Studio 将永远对我们的社区用户免费**，并且我们将在每个计划中提供许多未来的功能。请求功能并查看我们在[路线图](https://feedback.stately.ai)上的计划。

:::studio

我们在 Stately Studio [专业](studio-pro-plan.mdx)、[团队](studio-team-plan.mdx)或[企业](studio-enterprise-plan.mdx)计划中提供**免费试用**，以便您探索我们的高级功能如何为您和您的团队服务。您可以在登录 Stately Studio 时[升级](/upgrade.mdx)。

:::

## 社区功能

我们的社区计划功能包括以下内容：

- 无限的[公共项目](projects.mdx#change-a-projects-visibility)
- [设计和模拟流程](/studio.mdx#studio-editor)
- [从代码导入](import-from-code.mdx)
- [导出为代码](export-as-code.mdx)
- [发现和分叉公共机器](discover.mdx)
- [XState VS Code 扩展](xstate-vscode-extension.mdx)

:::info

## 为什么要支付高级计划费用？

我们的高级计划包括私有和未列出的项目、Stately AI 的 AI 辅助、Stately Sky 的实时工作流、团队管理的有用功能等等。您的订阅还帮助支持 Stately 团队构建和改进 Stately Studio 和 XState。

:::
