---
title: 键盘快捷键
---

Stately Studio 有许多键盘快捷键。

<Tabs>
<TabItem value="macos" label="macOS">

| 键盘快捷键                                          | 描述                |
| ---------------------------------------------------- | ------------------- |
| <kbd>Backspace</kbd>                                 | 删除                |
| <kbd>Command</kbd> + <kbd>C</kbd>                    | 复制                |
| <kbd>Command</kbd> + <kbd>V</kbd>                    | 粘贴                |
| <kbd>Command</kbd> + <kbd>Z</kbd>                    | 撤销                |
| <kbd>Command</kbd> + <kbd>Shift</kbd> + <kbd>Z</kbd> | 重做                |
| <kbd>Esc</kbd>                                       | 关闭面板或菜单      |
| <kbd>+</kbd>                                         | 放大                |
| <kbd>-</kbd>                                         | 缩小                |
| <kbd>Shift</kbd> + <kbd>0</kbd>                      | 缩放到 100%         |
| <kbd>Shift</kbd> + <kbd>1</kbd>                      | 适应内容视图        |
| <kbd>Arrow up</kbd>                                  | 向上平移            |
| <kbd>Shift</kbd> + <kbd>Arrow up</kbd>               | 大幅向上平移        |
| <kbd>Arrow down</kbd>                                | 向下平移            |
| <kbd>Shift</kbd> + <kbd>Arrow down</kbd>             | 大幅向下平移        |
| <kbd>Arrow left</kbd>                                | 向左平移            |
| <kbd>Shift</kbd> + <kbd>Arrow left</kbd>             | 大幅向左平移        |
| <kbd>Arrow right</kbd>                               | 向右平移            |
| <kbd>Shift</kbd> + <kbd>Arrow right</kbd>            | 大幅向右平移        |
| <kbd>Spacebar</kbd> + 鼠标按下                       | 拖动以平移          |
| <kbd>Command</kbd> + <kbd>.</kbd>                    | 显示/隐藏 UI        |

</TabItem>
