title: 使用图像 URL 共享机器图像
---

您可以在任何支持图像的地方共享您的机器图像。您可以使用图像 URL 来实现实时更新的图像，其中机器始终更新为您的最新更改。机器图像在文档中非常有用，包括 GitHub 拉取请求。

下面的图像是使用复制图像 URL 嵌入的。尝试在文档顶部栏中切换光明和黑暗模式；图像也会随之更改颜色模式。

您的机器图像仅在以下情况下可用：

- 项目可见性为 **公开** 或 **未列出**

机器图像不适用于私有机器。

阅读[如何更改项目的可见性设置](projects.mdx#change-a-projects-visibility)。

:::tip

您还可以[嵌入您的机器](embed.mdx)，以在 Stately Studio 的编辑器中获得专注的不可编辑视图。

:::

## 复制图像 URL

使用 <LinkIcon size={16} /> **复制图像 URL** 选项，从 <MoreHorizontal size={18} /> 三点图标旁边的机器名称中选择。

## 颜色模式

默认情况下，图像的颜色模式将与您选择的 Stately Studio 颜色模式相同。将 `.light.png` 或 `.dark.png` 添加到 URL 以强制使用该颜色模式。
