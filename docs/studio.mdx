你可以使用 [Stately Studio 的编辑器](/studio.mdx#stately-studios-editor) 通过状态机和状态图以可视化方式建模你的逻辑；无需编写代码！与同事和朋友通过[团队中的共享项目](/#projects-and-teams)协作。使用 [XState VS Code 扩展](xstate-vscode-extension.mdx) 在代码编辑器中与代码库一起使用编辑器，或将代码从 Stately Studio [导出](/#export) 到你的代码库中。

什么是状态机和状态图？我们很高兴你问了！[查看我们对状态机和状态图的无代码介绍](state-machines-and-statecharts.mdx)。

## Stately Studio 的编辑器

Stately Studio 的编辑器支持你需要的所有功能来可视化构建状态机和状态图。编辑器目前有两种模式；用于创建机器的 [**设计** 模式](design-mode.mdx) 和用于模拟机器工作方式的 [**模拟** 模式](simulate-mode.mdx)。

:::video

[观看我们的快速介绍视频 (7分36秒) 了解如何使用 Stately](https://www.youtube.com/watch?v=EzYIerEutgk)。

:::

### 快速入门教程

你可以从编辑器中的蓝色 <HelpCircle size={12} /> 帮助按钮访问快速入门教程。

当你第一次访问编辑器时，会显示两个简短的视频 (7分36秒) 作为快速入门指南。你可以随时从以下位置再次访问这些视频：

- 编辑器菜单 > 帮助 > <GraduationCap size={12} /> **学习 Stately**。
- <HelpCircle size={12} /> 帮助按钮 > <GraduationCap size={12} /> **学习 Stately**。
