## XState CLI（命令行界面）

[@xstate/cli（命令行界面）包](https://github.com/statelyai/xstate-tools/tree/master/apps/cli) 包含运行 typegen 的命令。该包目前很小，但我们计划添加更多功能。

### 安装

运行 `npm install @xstate/cli`。

### 命令

#### `xstate typegen <files>`

使用以下命令对一个 glob 运行 typegen。

`xstate typegen "src/**/*.ts?(x)"`

运行 typegen 将扫描每个目标文件并生成一个伴随的 typegen 文件。它还会将 typegen 导入到你的文件中，如 [我们的 typegen 文档](typegen.mdx) 中所述。

:::warning

确保将你的 glob 用引号括起来以确保正确执行。如果不将 glob 用引号括起来，它将被解释为文件列表，而不是 glob，这将导致意外的结果。

:::

#### 选项

`xstate typegen "src/**/*.ts?(x)" --watch`

在监视模式下运行任务，监视更改的文件并对其运行 typegen 脚本。
