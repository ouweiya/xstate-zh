您可以在任何支持 [`<iframe>`](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/iframe) 的地方嵌入您的机器。嵌入模式是 Stately Studio 编辑器中一个专注的不可编辑视图，可以缩放和平移。

我们喜欢在文档中使用嵌入的机器！下面是一个嵌入机器的示例：

<EmbedMachine
  name="嵌入功能机器"
  embedURL="https://stately.ai/registry/editor/embed/9e7f6b1e-416a-491f-a4a0-bba386a02d8d?mode=Design&machineId=e94d5674-7ab7-4a4e-b7a8-781609bfe523"
/>

您的机器只有在以下情况下才可以嵌入：

- 项目可见性为 **公开**
- 项目可见性为 **未列出**

私有机器无法使用嵌入 URL。

阅读[如何更改项目的可见性设置](/docs/projects/#change-a-projects-visibility)。

:::tip

您还可以[将您的机器作为图像获取](image.mdx)，以便在不支持 `<iframe>` 的平台上使用。

:::

## 复制嵌入 URL

1. 使用 Stately Studio 顶部栏中的 **分享** 按钮打开分享对话框。
2. 使用 **复制嵌入 URL** 按钮将机器的嵌入 URL 复制到剪贴板。
3. 在 `<iframe>` 中使用嵌入 URL，以便在任何支持 iframes 的地方嵌入机器。[在下面找到一些示例代码](#example-iframe-code)。

## 示例 `<iframe>` 代码

以下 HTML 在页面顶部以暗模式嵌入相同的机器：

```html
<iframe
  loading="lazy"
  src="https://stately.ai/registry/editor/embed/9e7f6b1e-416a-491f-a4a0-bba386a02d8d?mode=Design&machineId=e94d5674-7ab7-4a4e-b7a8-781609bfe523&colorMode=dark"
  style="display: block; width: 100%; aspect-ratio: 6 / 4;"
>
  <a
    href="https://stately.ai/registry/editor/embed/9e7f6b1e-416a-491f-a4a0-bba386a02d8d?mode=Design&machineId=e94d5674-7ab7-4a4e-b7a8-781609bfe523"
  >
    在 Stately Studio 查看 <em>嵌入功能机器</em> 机器 </a
  >.
</iframe>
```

:::note

如果您希望我们优先改进嵌入模式，请在我们的反馈页面上[投票](https://github.com/statelyai/feedback/issues/94)。

## URL 参数，包括颜色模式

嵌入 URL 具有与[机器 URL](url.mdx) 相同的一些参数。

- **machineId**：机器的唯一 ID。例如，`machineId=491a4c60-5300-4e22-92cf-8a32a8ffffca`
- **mode**：当前机器模式。例如，`mode=Design` 或 `mode=Simulate`
- **colorMode**：嵌入机器的颜色模式。例如，`colorMode=light` 或 `colorMode=dark`

默认情况下，颜色模式将与您选择的 Stately Studio 颜色模式相同。添加 `&colorMode=light` 或 `&colorMode=dark` 到 URL 以强制使用该颜色模式。
