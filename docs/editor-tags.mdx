# 标签

你可以在 Stately 编辑器中为状态添加标签：

- 选择你要标记的状态并使用 <Plus size={18} /> **标签**。使用文本输入框输入标签名称。
- 选择你要标记的状态，打开状态 <Info size={18} /> **详细信息** 面板，并使用 <Plus size={18} /> **标签** 按钮。使用文本输入框输入标签名称。
- 使用 <PlusSquare size={18} /> 加号图标按钮，在最近的标签旁边添加更多标签。
