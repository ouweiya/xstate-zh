---
title: 生成 React 应用
---

**生成 React 应用** 是一个实验性功能，可以帮助你从状态机生成一个基本的 React 应用。

在 <Code size={18} /> **代码** 面板中，使用 **生成 React 应用** 生成运行 React 应用所需的文件。你可以从生成的文件中复制代码，或者使用 **打开沙盒** 在 CodeSandbox 中打开生成的 React 应用。你还可以[使用 **增强应用** 按钮](#enhance-app)生成一个使用 Stately AI 的 React UI。

生成 React 应用生成的代码适用于 [XState V5](xstate.mdx)。[阅读如何从 XState V4 迁移到 V5](migration.mdx)。

:::tip

社区用户每月可以尝试有限次数的 **生成 React 应用**。 [升级你的现有计划](https://stately.ai/registry/billing)。

:::

:::studio

**生成 React 应用** 是 Stately Studio 的高级功能。你可以通过免费试用来尝试 Stately Studio 的高级计划。[查看我们的专业计划功能](studio-pro-plan.mdx)、[团队计划](studio-team-plan.mdx)、[企业计划](studio-enterprise-plan.mdx) 或 [升级你的现有计划](https://stately.ai/registry/billing)。

:::

你每月可以使用有限次数的生成。可用的生成次数在每月初重置。

## 增强应用

在生成 React 应用后，使用 **增强应用** 按钮生成一个增强的用户界面。生成后，模态窗口右上角的下拉菜单提供了预览增强应用的三个不同版本以及增强前生成的默认应用的选项。

:::tip

当你使用 **增强应用** 时，你的状态机和生成的 React 应用会发送到 OpenAI 以生成增强的应用。

:::
