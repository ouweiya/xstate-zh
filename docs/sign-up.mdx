# 注册

[从注册页面注册一个 Stately Studio 账户](https://stately.ai/registry/login)或使用[编辑器](https://stately.ai/editor)顶部栏中的**登录**按钮。

使用您的电子邮件地址和密码注册，或使用您的 GitHub、Google 或 Twitter 账户凭据登录。

:::studio

我们为 Stately Studio 的[专业版](studio-pro-plan.mdx)、[团队版](studio-team-plan.mdx)或[企业版](studio-enterprise-plan.mdx)计划提供**免费试用**，以便您和您的团队探索我们的高级功能。登录 Stately Studio 后，您可以[升级](/upgrade.mdx)。

:::

### 使用电子邮件

您现在可以使用电子邮件地址和密码注册一个 Stately Studio 账户。

#### 忘记密码

如果您忘记或想要更改密码，请使用[登录页面](https://stately.ai/registry/login)上的忘记密码链接。

### 使用社交账号登录

您也可以[使用您的 GitHub、Google 或 Twitter 账户凭据登录 Stately Studio](https://stately.ai/registry/login)。由于您已经在选择的平台上注册，您可以跳过 Stately Studio 的注册步骤，直接登录。

当您使用 GitHub、Google 或 Twitter 登录时，我们会将有关您的账户信息以及团队和项目邀请的电子邮件发送到您在该平台上注册时使用的电子邮件地址。
