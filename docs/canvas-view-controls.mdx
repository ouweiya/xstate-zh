## 视图控件

在设计和模拟状态机时，您会在画布上花费大量时间，因此我们添加了视图控件来帮助您导航您的机器。

## 显示/隐藏 UI（用户界面）

您可以显示和隐藏大多数编辑器的用户界面，以帮助您专注于您的机器。

1. 从 Stately Studio 左上角的 Stately 图标打开编辑器菜单。
2. 从 **视图** 子菜单中，切换 <Check size={18} /> **显示/隐藏 UI** 以显示和隐藏 UI。

您还可以使用 <kbd>Command</kbd>/<kbd>Ctrl</kbd> + <kbd>.</kbd> 键盘快捷键来显示和隐藏 UI。

:::tip

我们为许多视图控件提供了[键盘快捷键](keyboard-shortcuts.mdx)。

:::

## 视图控件

- <Undo size={18}/> 撤销
- <Redo size={18}/> 重做
- <Scan size={18}/> 居中视图：将当前选定的项目或整个机器居中显示。
- 缩放 %：当前缩放级别。按下以打开缩放菜单。

:::studio

[了解更多关于用户偏好中的浅色模式、深色模式和透明度的信息](user-preferences.mdx)。

:::

## 缩放到选定内容

您可以在画布上或 <ListTree size={18} /> **结构** 面板中右键单击任何转换或状态，使用 **缩放到选定内容**。
