注释使用 Markdown 格式，因此您可以在注释中添加链接、图片和其他格式。

:::tip

与[**描述**](descriptions.mdx)不同，注释不需要连接到状态或转换，可以在机器内部的任何位置添加或放置。

:::

## 向机器添加注释

您可以在**设计**和**模拟**模式下的机器上任何位置添加注释，但只能在**设计**模式下编辑它们。

注释与状态或转换无关。但在创建它们时，它们的布局位置与最近的根状态或父状态连接，这有助于在自动布局中为您的注释提供足够的空间。

- 使用画布工具中的 <PlusSquare size={18}/> 加号图标按钮并选择**注释**。
- 右键单击画布上的任意位置并选择**添加注释**。

### 编辑和删除注释

- 双击或使用 <kbd>Enter</kbd> 键编辑注释内容。
- 使用 <kbd>Backspace</kbd> 键删除选定的注释。
