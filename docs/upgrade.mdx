# 升级

**Stately Studio 将永远对我们的社区用户免费**，并且我们会在每个计划中提供许多未来的功能。

我们在 Stately Studio 的 [专业版](studio-pro-plan.mdx)、[团队版](studio-team-plan.mdx) 或 [企业版](studio-enterprise-plan.mdx) 计划上提供**免费试用**，以便你可以探索我们的高级功能如何为你和你的团队工作。当你登录 Stately Studio 时，你可以[升级](upgrade.mdx)。

## 升级到专业版或团队版

当你登录 Stately Studio 时，可以使用编辑器底部的 **升级** 按钮进行[升级](upgrade.mdx)。

## 升级到企业版

<p><a href="mailto:support@stately.ai?subject=I'm interested in Stately Studio Enterprise plan">发送邮件给 Stately 团队</a>，以获取根据你组织需求定制的计划。</p>

## 管理你的订阅计划

- 在编辑器中，使用编辑器菜单，并在 **账户** 部分选择 <CircleDollarSign size={18}/> **账单**。
- 在 Studio 的其他地方，从头部选择你的头像，并从账户菜单中选择 <CircleDollarSign size={18}/> **账单**。
