# 资产

你可以将资产拖放到任何状态上，或使用 <Plus size={18} /> 加号菜单和 <Paperclip size={18}/> **资产** 上传到任何选定的状态。

:::studio

资产是 Stately Studio 的高级功能。你可以通过免费试用来体验 Stately Studio 的高级计划。[查看我们专业计划的功能](studio-pro-plan.mdx)，[团队计划](studio-team-plan.mdx)，[企业计划](studio-enterprise-plan.mdx)或[升级你现有的计划](https://stately.ai/registry/billing)。

:::

当你想在状态机中包含用例、功能、工作流程等时，资产非常有价值。我们最常收到的请求之一是来自希望将其逻辑与实际用户界面联系起来的团队，资产提供了一种将用户界面设计与逻辑连接起来的方式，使你的整个团队都能理解。

:::new

你现在可以[嵌入 Figma 框架](figma.mdx)，这些框架会与 Figma 文件保持同步。

:::

## 资产顺序

第一个资产由 <Star size={18}/> 星形图标表示，是在画布上显示的资产。

更多资产可以从状态 <Info size={18}/> **详细信息** 面板中访问，你还可以在其中拖动以重新排序资产并选择其[显示大小](#asset-sizes)。

## 资产大小

一旦你的资产上传到状态，你可以从悬停在资产上的 **大小** 菜单或状态 <Info size={18}/> **详细信息** 面板中的下拉菜单中选择显示大小：

- 小 **sm**
- 中 **md**
- 大 **lg**
- 特大 **xl**

:::tip

你还可以使用 markdown 从任何 URL 添加图像到[状态和转换描述](descriptions.mdx)中。

:::
